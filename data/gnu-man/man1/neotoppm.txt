NAME
       neotoppm - convert an Atari Neochrome .neo into a portable pixmap

SYNOPSIS
       neotoppm [neofile]

DESCRIPTION
       Reads an Atari Neochrome .neo file as input.  Produces a portable pixmap as output.

SEE ALSO
       ppmtoneo(1), ppm(5)
