NAME
       mkindex - script to process LaTeX index and glossary files

SYNOPSIS
       mkindex BASENAME[.idx] ...

DESCRIPTION
       mkindex  runs  makeindex(1)  on  any  index  (.idx) or glossary (.glo) files created by TeXing the file BASE‚Äê
