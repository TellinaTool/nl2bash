NAME
       gucharmap - Unicode character picker and font browser

SYNOPSIS
       gucharmap [OPTION...]

DESCRIPTION
       gucharmap  allows you to browse through all the available Unicode characters and categories for the installed
       fonts, and to examine their detailed properties.

OPTIONS
       --font Font to start with, ex: 'Serif 27'

       --display=DISPLAY
              X display to use

       gucharmap also accepts the standard GNOME and GTK options.
