NAME
       hciemu - HCI emulator

SYNOPSIS
       hciemu [options] local_address


DESCRIPTION
       hciemu is used to emulate an HCI via hci_vhci kernel module


OPTIONS
       -d device
              use specified device

       -b bdaddr
              emulate bdaddr

       -s file
              create snoop file file

       -n     do not detach
