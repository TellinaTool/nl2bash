NAME
       atktopbm - convert Andrew Toolkit raster object to portable bitmap

SYNOPSIS
       atktopbm [atkfile]

DESCRIPTION
       Reads an Andrew Toolkit raster object as input.  Produces a portable bitmap as output.

SEE ALSO
       pbmtoatk(1), pbm(5)
