NAME
       pbmtoatk - convert portable bitmap to Andrew Toolkit raster object

SYNOPSIS
       pbmtoatk [pbmfile]

DESCRIPTION
       Reads a portable bitmap as input.  Produces a Andrew Toolkit raster object as output.

SEE ALSO
       atktopbm(1), pbm(5)
