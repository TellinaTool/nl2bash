NAME
       manhole - Connect to a Twisted Manhole service

SYNOPSIS
       manhole

DESCRIPTION
       manhole  is  a GTK interface to Twisted Manhole services. You can execute python code as if at an interactive
       Python console inside a running Twisted process with this.
