NAME
       ppmtoneo - convert a portable pixmap into an Atari Neochrome .neo file

SYNOPSIS
       ppmtoneo [ppmfile]

DESCRIPTION
       Reads a portable pixmap as input.  Produces an Atari Neochrome .neo file as output.

SEE ALSO
       neotoppm(1), ppm(5)
