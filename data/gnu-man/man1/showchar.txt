NAME
       showchar - show character with information

SYNOPSIS
       showchar [ fontname ] [ charname ]

DESCRIPTION
       This manual page was written for the Debian GNU/Linux distribution because the original program does not have
       a manual page.

       showchar outputs PostScript to draw a character with metric info.
