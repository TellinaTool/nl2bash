NAME
       bluez-simple-agent - A PIN management and agent program for pairing to Bluetooth device.

SYNOPSIS
       bluez-simple-agent [<hciX>] [<bdaddr>] [remove]

DESCRIPTION
       bluez-simple-agent is pass agent program for bluetooth.

OPTIONS
       <hciX>
           The command is applied to device hciX , which must be the name of an installed Blue‐ tooth device. If not
           specified, the command will be sent to the first available Blue‐ tooth device.

       <bdaddr>
           bdaddr of device doing pairing.

       remove
           Remove intended bdaddr from database.
