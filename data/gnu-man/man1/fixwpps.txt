NAME
       fixwpps - filter to fix WP documents so PSUtils work

SYNOPSIS
       fixwpps < WordPerfect.ps > Fixed.ps

DESCRIPTION
       Fixwpps  is  a  perl  filter which "fixes" PostScript from WordPerfect 5.0 and 5.1 so that it works correctly
       with Angus Duggan's psutils package.
