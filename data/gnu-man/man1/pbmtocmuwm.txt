NAME
       pbmtocmuwm - convert a portable bitmap into a CMU window manager bitmap

SYNOPSIS
       pbmtocmuwm [pbmfile]

DESCRIPTION
       Reads a portable bitmap as input.  Produces a CMU window manager bitmap as output.

SEE ALSO
       cmuwmtopbm(1), pbm(5)
