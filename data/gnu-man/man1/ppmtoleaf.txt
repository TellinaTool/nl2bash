NAME
       ppmtoleaf - convert portable pixmap to Interleaf image format

SYNOPSIS
       ppmtoleaf [ppmfile]


DESCRIPTION
       Reads an Interleaf image file as input.  Generates a portable pixmap (PPM file) as output.
