NAME
       gouldtoppm - convert Gould scanner file into a portable pixmap

SYNOPSIS
       gouldtoppm [gouldfile]

DESCRIPTION
       Reads a file produced by the Gould scanner as input.  Produces a portable pixmap as output.

SEE ALSO
       ppm(5)
