NAME
       msexpand - decompress data compressed using mscompress(1) or COMPRESS.EXE

SYNOPSIS
       msexpand [ -h ]
       msexpand [ -V ]
       msexpand [ name ...  ]

DESCRIPTION
       Msexpand  decompress files compressed using mscompress(1) or Microsoft COMPRESS.EXE utility (e.g. Windows 3.x instal‚Äê
       lation files). If no files are specified, the standard input is decompressed to the standard output.
