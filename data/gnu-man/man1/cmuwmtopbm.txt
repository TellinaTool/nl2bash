NAME
       cmuwmtopbm - convert a CMU window manager bitmap into a portable bitmap

SYNOPSIS
       cmuwmtopbm [cmuwmfile]

DESCRIPTION
       Reads a CMU window manager bitmap as input.  Produces a portable bitmap as output.

SEE ALSO
       pbmtocmuwm(1), pbm(5)
