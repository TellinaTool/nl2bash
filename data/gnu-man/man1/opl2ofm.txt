NAME
       opl2ofm - convert Omega and TeX property-list files to font-metric files

SYNOPSIS
       opl2ofm [OPTIONS] OPLFILE[.opl] [OFMFILE[.ofm]]

DESCRIPTION
       opl2ofm  translates a property-list format file, OPLFILE, into the binary Omega Font Metric format.  The pro‚Äê
       gram writes to standard output (by default) or to a file specified as OFMFILE.

       The program also works with TeX PL files, producing TeX TFM files.  (opl2ofm is based on the WEB source  code
       for pltotf(1).)

OPTIONS
       -help  display a brief summary of syntax and options

       -verbose
              display progress reports

       -version
              output version information and exit

BUGS
       None  known,  but report any found to <omega@ens.fr> (mailing list).  You should also check to see if the bug
       is present in pltotf(1).
