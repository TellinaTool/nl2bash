NAME
       remmina - remote desktop client for GNOME desktop environment

SYNOPSIS
       remmina

DESCRIPTION
       Remmina  is  a  remote desktop connection client able to display and control a remote desktop session. It can
       connect to a VNC platform as well as windows terminal servers.
