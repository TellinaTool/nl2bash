NAME
     icontool -- tool for testing and debugging IconServices.

SYNOPSIS
     icontool [-abcd] [-a path] [file] [file ...] arg0 arg2 ...

DESCRIPTION
     icontool a tool for testing and debugging IconServices. Allows the user
     to dump the cache content, clear the cache, etc.

FILES
     /usr/local/bin/icontool  executable
