NAME
     mdmclient -- MDM (Mobile Device Management) client

DESCRIPTION
     mdmclient

     Used internally for communication with Mobile Device Management (Profile
     Manager) server. Receives configuration profiles and commands from the
     server after a user binds to the server by installing a binding profile.
     Part of Managed Client (MCX).
