NAME
     afconvert -- Audio File Convert

SYNOPSIS
     afconvert [option] src_audiofile dest_audiofile

DESCRIPTION
     Audio File Convert will convert a source audio file to a new audio file
     with the specified file and data types

OPTIONS
     -h       print help text
