
USERS(1)                  BSD General Commands Manual                 USERS(1)

NAME
     users -- list current users

SYNOPSIS
     users

DESCRIPTION
     users lists the login names of the users currently on the system, in
     sorted order, space separated, on a single line.

FILES
     /var/run/utmpx

SEE ALSO
     finger(1), last(1), who(1), utmpx(5)

HISTORY
     The users command appeared in 3.0BSD.
